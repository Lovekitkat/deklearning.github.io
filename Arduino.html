กลับหน้าหลัก บทเรียนที่ 2: อุปกรณ์ Arduino

การเรียนรู้พื้นฐานการเชื่อมต่อพอร์ตและการทำงานของ GPIO บน Arduino

ศึกษาโครงสร้างขาอินพุตและเอาต์พุต (GPIO) ของบอร์ด Arduino พร้อมแนวทางการเชื่อมต่อเซนเซอร์และโมดูลต่าง ๆ

การเชื่อมต่อพอร์ตของบอร์ด Arduino UNO

เริ่มต้นเรียนรู้การใช้งานขาพอร์ตและเซนเซอร์เบื้องต้นสำหรับ Maker มือใหม่

GPIO คืออะไร?

GPIO (General Purpose Input/Output) คือ ขาพอร์ตเอนกประสงค์บนบอร์ด Arduino ที่เราสามารถเขียนโปรแกรมสั่งการได้ ว่าจะให้ทำหน้าที่เป็น Input (รับค่าจากสวิตช์หรือเซนเซอร์) หรือเป็น Output (ส่งไฟออกไปสั่งงาน เช่น เปิดหลอดไฟ หรือขับมอเตอร์)

ภาพที่ 1: บอร์ด Arduino UNO R3 ที่นิยมใช้งานมากที่สุด

ประเภทพอร์ตของ Arduino UNO

บอร์ด Arduino UNO มีการจัดกลุ่มพอร์ตตามหน้าที่การใช้งานดังนี้:

ภาพที่ 2: ไดอะแกรมแสดงชื่อขาพอร์ต (Pinout Diagram)

1. Power Pins (กลุ่มพอร์ตจ่ายไฟ)

  • 5V & 3.3V: จ่ายไฟเลี้ยงแรงดันคงที่ให้กับโมดูลต่าง ๆ
  • GND (Ground): จุดต่อสายดิน ขั้วลบที่ทุกอุปกรณ์ต้องใช้ร่วมกัน
  • VIN: ใช้สำหรับจ่ายไฟเข้าบอร์ดจากแหล่งจ่ายภายนอก (7-12V)

2. Digital Pins (D0 - D13)

ใช้รับหรือส่งสัญญาณแบบ HIGH / LOW (ติด/ดับ) เท่านั้น เหมาะกับปุ่มกดหรือไฟ LED

3. PWM Pins (3, 5, 6, 9, 10, 11)

พอร์ตที่มีสัญลักษณ์ ~ หน้าตัวเลข สามารถส่งค่าความแรงของไฟแบบ "จำลองอนาล็อก" ได้ เช่น การหรี่ไฟ หรือควบคุมองศาของ Servo Motor

4. Analog Pins (A0 - A5)

ใช้สำหรับอ่านค่าสัญญาณที่เปลี่ยนแปลงต่อเนื่อง เช่น ค่าจากตัวต้านทานปรับค่าได้ หรือเซนเซอร์วัดความชื้น โดยจะแปลงเป็นตัวเลข 0 - 1023

ตัวอย่างการเชื่อมต่อกับเซนเซอร์

1. ปุ่มกด (Button)

เชื่อมต่อกับ Digital Pin เพื่อเช็คสถานะการกดสวิตช์

2. LDR (เซนเซอร์วัดความเข้มแสง)

เชื่อมต่อกับ Analog Pin (A0) เพื่ออ่านค่าความสว่างในห้องออกมาเป็นตัวเลข

3. Ultrasonic Sensor (เซนเซอร์วัดระยะทาง)

ต้องใช้ Digital Pin 2 ช่อง คือ Trig (ส่งคลื่น) และ Echo (รับคลื่น) เพื่อคำนวณเวลาที่คลื่นสะท้อนกลับมาเป็นระยะทาง

ตัวอย่างโค้ด Arduino

โค้ดอ่านค่าจาก LDR และสั่งเปิดไฟ LED บนบอร์ด (ขา 13) เมื่อแสงน้อย:


int ldrPin = A0; // ต่อ LDR ที่ช่อง A0

int ldrPin = A0; // ต่อ LDR ที่ช่อง A0
int ledPin = 13; // ใช้ LED บนบอร์ดที่ช่อง 13
int lightVal = 0; // ตัวแปรเก็บค่าแสง

void setup() {
    pinMode(ledPin, OUTPUT); // กำหนดขา 13 เป็นเอาต์พุต
    Serial.begin(9600); // เริ่มการสื่อสารกับคอมพิวเตอร์
}

void loop() {
    lightVal = analogRead(ldrPin); // อ่านค่าแสง (0-1023)
    Serial.println(lightVal); // แสดงค่าผ่าน Serial Monitor

    if (lightVal < 500) { // ถ้าค่าแสงน้อยกว่า 500
        digitalWrite(ledPin, HIGH); // เปิดไฟ
    } else {
        digitalWrite(ledPin, LOW); // ปิดไฟ
    }
    delay(100);
}
