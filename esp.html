🌐 Internet of Things (IoT) คืออะไร?

Internet of Things (IoT) คือ เครือข่ายของอุปกรณ์ต่างๆ เช่น เซนเซอร์, เครื่องใช้ไฟฟ้า หรืออุปกรณ์อิเล็กทรอนิกส์ ที่สามารถเชื่อมต่อและรับส่งข้อมูลระหว่างกันได้ผ่านอินเทอร์เน็ต โดยที่ไม่ต้องใช้มนุษย์มาควบคุมโดยตรงตลอดเวลา ช่วยให้เราสามารถสั่งการหรือตรวจสอบสถานะของอุปกรณ์จากที่ไหนก็ได้ในโลก

GPIO Pins น้อย (ประมาณ 11 ขา) มาก (ประมาณ 25-30 ขา)

🔌 โครงสร้างพอร์ตและขา (Pinout)

การเข้าใจหน้าที่ของแต่ละขาเป็นเรื่องสำคัญมาก เพื่อป้องกันความเสียหายของบอร์ด:

GPIO (General Purpose INPUT/OUTPUT) ขาอเนกประสงค์ ใช้รับค่าจากเซนเซอร์ (Input) หรือสั่งงานอุปกรณ์ (Output)
Digital Pins รับ/ส่งสัญญาณ 2 สถานะ คือ HIGH (3.3V) และ LOW (0V)
Analog Pins (ADC) ใช้รับค่าแรงดันต่อเนื่อง (เช่น 0V - 3.3V) มาแปลงเป็นตัวเลข (0-1023 หรือ 0-4095)
PWM (Pulse Width Modulation) จำลองแรงดันเอาต์พุต เช่น ใช้หรี่ไฟ LED หรือควบคุมความเร็วพัดลม
Power Pins - 3.3V: ไฟเลี้ยงหลัก
- 5V/VIN: ไฟเลี้ยงจาก USB หรือแหล่งจ่ายนอก
- GND: กราวด์ (ต้องต่อร่วมกันทุกอุปกรณ์)
⚠️ ข้อควรระวังเรื่องแรงดัน:
บอร์ด ESP32 และ ESP8266 ทำงานที่แรงดัน 3.3V เท่านั้น การป้อนไฟ 5V เข้าไปที่ขา GPIO โดยตรงอาจทำให้ชิปพังทันที!
🚩 ขาที่ควรระวัง (Boot Pins):
บางขา เช่น GPIO 0, GPIO 2 ใน ESP8266 จะส่งผลต่อการ Boot เครื่อง หากเราต่อเซนเซอร์ที่ดึงไฟจากขาเหล่านี้ขณะเปิดเครื่อง อาจทำให้บอร์ดไม่ทำงาน (Upload code ไม่เข้า)

🛠️ การเชื่อมต่อโมดูลตัวอย่าง

ปุ่มกด (Button)

ต่อขาข้างหนึ่งเข้า GPIO และอีกข้างเข้า GND (ควรใช้ Internal Pull-up)

DHT11 / DHT22

เซนเซอร์วัดอุณหภูมิและความชื้น เชื่อมต่อผ่านขา Digital (1 เส้น)

Ultrasonic

วัดระยะทาง ใช้ขา Trig (ส่ง) และ Echo (รับ) เป็นสัญญาณ Digital

LDR

เซนเซอร์วัดความเข้มแสง ต้องต่อเข้ากับขา Analog (ADC)

Example Code: อ่านค่าปุ่มและสั่งเปิดไฟ LED (Arduino IDE) C++ / Arduino 
const int buttonPin = 4; // ขาต่อปุ่มกด
const int ledPin = 2; // ขา LED ในตัวบอร์ด (Built-in)

void setup() {
  pinMode(ledPin, OUTPUT);
  pinMode(buttonPin, INPUT_PULLUP);

}

void loop() {
  int buttonState = digitalRead(buttonPin);

  if (buttonState == LOW) {
    digitalWrite(ledPin, HIGH);  // เมื่อปุ่มถูกกด (ดึงลงกราวด์)

  } else {
    digitalWrite(ledPin, LOW);
  }
}

สรุปเนื้อหา: การเริ่มต้น IoT สิ่งสำคัญคือการเลือกบอร์ดให้เหมาะกับงาน (ESP8266 สำหรับงานเล็ก, ESP32 สำหรับงานประสิทธิภาพสูง) และตรวจสอบตำแหน่ง Pinout ให้ถูกต้องเสมอก่อนเริ่มการเชื่อมต่ออุปกรณ์จริง
